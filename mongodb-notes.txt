Finding Arrays:

- db.movies.find({genres:"Drama"}) // returns documents like this  {...,genres: [ 'Drama', 'Horror', 'Thriller' ],...} // looks for all documents which has "Drama" in the array
- db.movies.find({genres:["Drama"]}) // returns documents like this   {...,genres: [ 'Drama' ],...}


- db.movies.find({genres:"Drama"},{"genres.$":1}) // second entry is projection  // "genres.$" - returns first match

- db.movies.find({genres:{$all:["Drama","Horror"]}},{"genres.$":1}) // $all is document with both "Drama" and "Horror", returns matched genre and object id and no else fields from the document


// Filtering your array or any document
- db.movies.find({genres:"Drama"},{genres:{$elemMatch:{$eq:"Horror"}}}) // filtering then projecting first get all documents with "Drama" inside the genres array in the document
// and then project documents containing "Horror"

basic syntax of $elemMatch operator

db.collection.find({
  <arrayField>: {
    $elemMatch: {
      <condition1>,
      <condition2>,
      ...
    }
  }
})

Example

      { "rating": 5, "reviewer": "John" },

    ]
  }
]

db.products.find({
  reviews: {
    $elemMatch: {
      rating: 5,
      reviewer: "Jane"
    }
  }
})

Using $elemMatch as a Projection Operator

db.products.find(
  { item: "Laptop", "reviews.reviewer": "John" },
  { "reviews.$elemMatch": { reviewer: "John" } }
)

The $elemMatch operator can also be used in the projection part of a find() operation to return only the first matching element of the array field, rather than the entire array. 
This query returns the "Laptop" document but limits the reviews array to contain only the first review where the reviewer is "John".
